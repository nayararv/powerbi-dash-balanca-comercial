# 📊 Dashboard da Balança Comercial Brasileira - Power BI

## 🎯 Objetivo
Analisar desempenho de vendas, faturamento, ticket médio e performance por região.

## 🛠 Ferramentas Utilizadas
- Power BI
- DAX
- SQL (tratamento de dados)
- Modelagem estrela

## 📌 Principais Insights
- Região Sudeste representa maior faturamento
- Produto X possui maior margem
- Sazonalidade no 4º trimestre
